# Python for C# Developers

A structured Python learning guide written from the perspective of an experienced C# developer. Covers syntax differences, idioms and production patterns.

This is not a beginner Python guide. It assumes strong backend engineering experience in C# and .NET and maps Python concepts directly to their C# equivalents.

## Modules

| # | Module | C# Equivalent |
|---|---|---|
| 01 | [Syntax Basics](./01-syntax-basics/) | Variables, loops, functions, type hints vs static typing |

| 03 | [OOP](./03-oop/) | Classes, dataclasses vs records, properties, interfaces |
| 04 | [Error Handling](./04-error-handling/) | try/except vs try/catch, custom exceptions |

## How to use this repo

Each module contains:
- A markdown cheatsheet comparing Python and C# side by side
- Working code examples
- Common mistakes to avoid when coming from C#
